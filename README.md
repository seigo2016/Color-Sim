<h1>糸かけ曼荼羅色シミュレータ</h1>

糸かけ曼荼羅の色をシミュレーションするソフトです  
対応環境… Web アプリケーション

## 機能一覧

### 色シミュレーション

### 形状 11+α 種類

-   円　 5 種類(ピン数…48,64,77,88,108)
-   正方形 1 種類(ピン数…64)
-   五角形 1 種類(ピン数…80)
-   六角形 1 種類(ピン数…72)
-   つぼみ
-   麻の葉
-   六芒星
-   ピン数と間隔をカスタマイズ出来る円
