<h1>糸かけ曼荼羅色シミュレータ</h1>   

これはタイトルの通り糸かけ曼荼羅の色をシミュレーションするソフトです  

対応機種…Mac (Linux) Windows   

<h2>機能一覧</h2>  
====================================================================  

<h3>色シミュレーション…各6種類</h1>  
円　5種類(ピン数…48,64,77,88,108)   
正方形 1種類(ピン数…64)  
五角形 1種類(ピン数…80)  
六角形 1種類(ピン数…72)   
つぼみ  
麻の葉  
六芒星
