糸掛曼荼羅色シミュレータ

これはタイトルの通り糸掛曼荼羅の色をシミュレーションするソフトです
対応機種…Mac Windows (Linux)

機能一覧
====================================================================
色シミュレーション…各6種類
・円　5種類(ピン数…48,64,77,88,108)
・正方形 1種類(ピン数…64)
・五角形 1種類(ピン数…80)
・六角形 1種類(ピン数…72)
